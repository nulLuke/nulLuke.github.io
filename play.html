Minecraft Java Edition

Chances are, you've added a server before on Minecraft Java Edition, but if not, here's a simple walkthrough of how to do it.

1. Click on "Multiplayer"
2. At the bottom select "Add Server"
3. In the "Server Address" field type in 'mc.lukesmp.net'
4. Enter whatever you want in the "Server Name" box
5. Click "Done"
6. The server should now be in your server list, simply select it and press "Join Server" 

Minecraft Bedrock Edition

This will only work on Minecraft for Windows 10 and Minecraft Pocket Edition (iOS or Android), if you play on a console, please refer to instructions on how to connect through a console edition

1. Click "Play" and go to the "Servers" tab
2. Scroll down and click "Add Server"
3. In the "Server Address" field type in the IP address: 'bedrock.lukesmp.net'
4. The "Port" field doesn't have to be changed because it defaults to '19132'
5. Of course enter a server name (this can be anything)
6. Click "Play" 

Minecraft for Consoles

On the Xbox, Playstation, and Nintendo Switch editions of Minecraft, you are not able to add custom servers for security reasons. Due to this you may not think that you're able to play on the LukeSMP but fear not, there's a workaround. By using "Bedrock Connect", which is a DNS server, you're able to connect and play on Minecraft servers not on Microsoft's predetermined list. Select which platform you play on below for a full tutorial on how to set it up.
