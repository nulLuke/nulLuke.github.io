Rules

In short, don't be a jerk

Notice

The punishment for breaking any of these rules tends to vary depending on the severity but generally it would be a temporary ban from the server.

Also keep in mind that the rules can be updated at any time as needed.

Rules 

  • No hacking
    No using clients that manipulate or assist your control (feel free to use Optifine, Badlion, or Lunar as long as there are no mods that allow for hacking).
  • No exploiting
    Using glitches to dupe or manipulate blocks/items how they are not intended to be used is against the rules, however TNT duplication is allowed as well as bedrock breaking.
  • No lava casting
    The only destruction of the landscape that ISN'T allowed is lava casting because they are annoying and no one likes them, it's as simple as that.
  • This is kinda a soft rule but don't be a jerk
    Nobody likes it when one person on the server just goes out of their way to ruin the experience for others, the server is made for people to have fun so if you want to be a jerk, I would suggest joining '2b2t.org' (the OLDEST ANARCHY SERVER IN MINECRAFT).
  • Have fun
    Don't be boring!
