4th time's a charm?

Season 4

The 4th world on LukeSMP is now live! It's a 1.18.x generated world hence taking advantage of the new terrain generation in the update "Caves and Cliffs Part 2". The world was then later updated to version 1.19.x which introduced the deep dark and Ancient Cities as well as the mangrove swamps (please note, you will have to travel to chunks not previously loaded in 1.18 to experience what this new update had to offer).

World Specifications

LukeSMP Season 4 is hosted on version 1.19.2, "Minecraft: The Wild Update", using PaperMC software.

This means the world will now generate with the new deep dark cave biome featuring the Ancient Cities as well as the mangrove swamps!

Running the server on software from PaperMC adds an extra performance and optimization boost allowing the server to be fast and responsive as well as secure!

Features

Cross Platform Support

Connect from either Java Edition or Bedrock Edition while maintaining the same player data on both!

Discord Integration

Talk to anyone online the server without even joining, the Minecraft chat is linked with discord!

Custom Map Art

Import any image from the internet to be rendered onto a map in-game, the possibilities are endless!
